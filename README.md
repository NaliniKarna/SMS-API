# SMS-API
A Dockerized FastAPI-based SMS forwarding system with template management, PostgreSQL integration, and secure API key authentication.
SMS Forwarding API is a production-ready backend service built using FastAPI, PostgreSQL, and Docker.

The system allows administrators to:

Create and manage SMS templates

Send SMS messages using stored templates

Authenticate requests using API keys

Log SMS delivery status in a PostgreSQL database

Deploy easily using Docker Compose

Features

RESTful API architecture

Template-based SMS management

API key authentication

PostgreSQL database integration

Dockerized deployment

SMS logging with gateway response tracking

UTF-8 support (including Nepali messages)

Tech Stack

Backend: FastAPI (Python)

Database: PostgreSQL

Containerization: Docker & Docker Compose

HTTP Client: Requests

SMS Gateway Integration
